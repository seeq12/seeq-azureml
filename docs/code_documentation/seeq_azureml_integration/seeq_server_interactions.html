class seeq.addons.azureml.backend._run_investigation.RunInvestigation(input_signals: dict, result_name: str, az_model_name: str, az_model_version: str, start: Union[pandas._libs.tslibs.timedeltas.Timedelta, datetime.datetime], end: Union[pandas._libs.tslibs.timedeltas.Timedelta, datetime.datetime], grid: Optional[str] = '5 min', workbook: Optional[str] = 'Data Lab >> Azure ML Integration', worksheet: Optional[str] = 'From Azure ML Integration', datasource: Optional[str] = 'Azure ML', endpoint_uri: Optional[str] = None, aml_primary_key: Optional[str] = None, self_signed_certificate=True, quiet=True)[source]ÔÉÅ

Bases: object

Takes the input parameters supplied by the user (typically, via Azure ML Integration UI), posts a request to the Azure ML model, gets a result signal back from Azure ML and pushes the result back to Seeq.

This class assumes that the Azure ML model returns only ONE signal.
